# Rochine Lab (Beta)

A live research lab for **verifiable automation** — experimenting with task validation, telemetry proofs, and reward flow for the Rochine network.

## Focus
- Verifiable task execution (digital & robotics workloads)
- Proof layer testing (artifact format, hashing, validator signatures)
- Autonomous robot handshake protocol (RobotLink trial)

## Environment
- Network: Solana **Devnet**
- Stage: **Active Beta**
- Updated: 2025-11-09
